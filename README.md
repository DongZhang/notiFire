# notiFire
notiFire is a simple library that providing message notification. No jQuery needed!!

### Examples
See example.html

### Usage
##### Simple message:
```
notifire({
    msg: 'this is a test message'
});
```

##### Other options:
```
notifire({
    msg: 'this is a test message',
    types: 'success',
    width: 400,
    height: 100,
    position: 'right',
    timeout: 1000
});
```
